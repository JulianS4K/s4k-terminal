HOW TO ADD API KEYS
API credentials are stored securely in GAS Script Properties — not in this browser.

1. Go to script.google.com → open your s4k-terminal project
2. Click Project Settings (gear icon, left sidebar)
3. Scroll to Script Properties → Add property:

SG_CLIENT_ID → your SeatGeek client ID
SG_CLIENT_SECRET → your SeatGeek secret
TE_API_KEY → your TEvo API key
TE_SECRET → your TEvo secret

4. Click Save script properties
5. Redeploy the GAS script (New deployment)
6. Paste the new URL above and hit Test Connection
